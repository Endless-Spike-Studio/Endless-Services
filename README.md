> 提供 Geometry Dash 国服, Geometry Dash 官服加速, Newgrounds 加速下载 等服务

### 手动部署

```shell
composer install --no-dev
pnpm install
php vendor/bin/envoy run deploy
php artisan octane:start # 使用 octane 启动
```
